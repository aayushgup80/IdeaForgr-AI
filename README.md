# IdeaForgr AI 

Hello Everyone!
This is an AI-powered startup idea workspace built with Gemini.

## Setup

1. Clone this repository or download the ZIP file and extract it.
2. Open Command Prompt/PowerShell in the folder and run npm install
3. Create a .env file with your Gemini API key:
    GEMINI_API_KEY = your_key_here
4. Get a free API key at aistudio.google.com
5. Run node server.js
6. Open http://localhost:3000
